# ChatGPT інтеграції для онлайн-торгівлі: що реально працює, а що — ні

Якщо ти продаєш на Rozetka, Prom чи через власний сайт — скоріш за все, вже чув, що "ChatGPT автоматизує все". І так само скоріш за все, спробував щось одне, не побачив магії, і закрив вкладку.

Ця стаття — для тих, хто хоче розібратися без ейфорії: де ChatGPT справді економить час і гроші, як його технічно підключити навіть без програміста, і де він підведе, якщо покладатися сліпо.

---

## Що взагалі можна автоматизувати за допомогою ChatGPT

Перш ніж лізти в інтеграції, варто чесно відповісти: а що у тебе зараз займає найбільше часу?

Ось найпоширеніші задачі українських онлайн-продавців, де ChatGPT дає реальний ефект:

- **Написання описів товарів** — особливо якщо SKU сотні або тисячі
- **Відповіді на типові запитання клієнтів** у чаті або email
- **Генерація контенту** для карток на маркетплейсах (заголовки, буллети, характеристики)
- **Обробка відгуків** — формування відповідей на негативні коментарі
- **Переклад описів** під різні ринки (наприклад, якщо виходиш на Amazon)
- **Аналіз конкурентів** — обробка великого обсягу тексту зі сторінок конкурентів

Де він **не допоможе без суттєвих зусиль**: аналітика продажів (він не бачить твоїх даних сам по собі), реальна комунікація з Новою Поштою чи митницею, і все, що вимагає актуальних даних — ціни, наявність, логістика.

## Три рівні інтеграції: від простого до складного

### Рівень 1. Без жодного коду — просто ChatGPT.com

Так, звучить банально. Але більшість ФОПів недовикористовують навіть базовий інструмент.

Що зробити прямо зараз:

1. Зайди на [chat.openai.com](https://chat.openai.com) або використовуй GPT-4 через платний акаунт ($20/міс)
2. Створи **Custom Instructions** або збережений промпт для свого магазину

Приклад промпту для продавця на Prom:

```
Ти — копірайтер інтернет-магазину жіночого одягу українського виробництва.
Пишеш описи товарів для Prom.ua. Тон — дружній, але без пафосу.
Завжди вказуй склад тканини, розмірну сітку та догляд за виробом.
Уникай слів "ексклюзивний", "унікальний", "преміум".
```

Це займає 10 хвилин, але економить годину щодня на рутинному контенті.

---

### Рівень 2. Make (Integromat) або Zapier — автоматизація без коду

Якщо хочеш, щоб ChatGPT реагував автоматично — наприклад, отримував новий відгук і одразу генерував відповідь — тут на допомогу приходять no-code платформи.

**Make.com** (раніше Integromat) — зручніший і дешевший варіант для більшості e-commerce задач. Є безкоштовний план.

**Що можна зробити:**

- Отримав новий відгук на Rozetka → через вебхук потрапляє в Make → ChatGPT генерує відповідь → ти отримуєш у Telegram і одним кліком відправляєш
- Нова заявка з форми на сайті → ChatGPT формує персоналізований лист → автоматично йде клієнту через Gmail або SendPulse

**Як підключити ChatGPT в Make:**

1. Реєструєшся на make.com
2. Створюєш новий сценарій (scenario)
3. Обираєш модуль **OpenAI (ChatGPT)** — він там вже є
4. Вводиш свій API ключ з platform.openai.com
5. Будуєш ланцюжок: тригер → OpenAI → дія

API коштує окремо від ChatGPT Plus. Орієнтовно: 1000 описів товарів по 200 слів — це ~$2-4 залежно від моделі. Для більшості малого бізнесу це копійки.

---

### Рівень 3. API напряму — для тих, хто має розробника

Якщо у тебе є власний сайт на WordPress, самописній платформі або ти інтегруєшся з Rozetka через їхній API — можна підключити OpenAI API напряму.

Це має сенс, коли:
- Обсяг задач великий (тисячі SKU)
- Потрібна кастомна логіка (наприклад, генерація опису з XML-фіду постачальника)
- Хочеш чат-бота прямо на своєму сайті

Орієнтовна вартість розробки простої інтеграції у фрилансера — 5000–15000 грн. Для магазину зі 100+ SKU на місяць це окупається швидко.

## Конкретний кейс: автоматизація описів для Rozetka

Уявімо типову ситуацію: постачальник прислав Excel з 300 новими позиціями. Назва, артикул, технічні характеристики — і все. Описів немає.

**Без ChatGPT:** копірайтер, 2-3 дні роботи, ~3000–6000 грн.

**З ChatGPT через Make:**

1. Завантажуєш Excel у Google Sheets
2. Make зчитує кожен рядок
3. Відправляє дані в ChatGPT з промптом типу: *"Напиши опис товару для Rozetka на основі цих характеристик. 150-200 слів, структурований текст, без води."*
4. Результат пишеться назад у Sheets
5. Ти перевіряєш і завантажуєш

Час налаштування: 2-3 години вперше. Час роботи на 300 позицій: 20-40 хвилин. Вартість: $3-5.

**Важливо:** автоматично згенеровані описи потребують перевірки. ChatGPT може вигадати характеристики, яких немає, або написати щось некоректне для специфічної ніші (наприклад, медичні товари, де точність критична). Завжди перечитуй або хоча б вибірково перевіряй.

## Чат-боти для обслуговування клієнтів: реально чи ні?

Окремо варто поговорити про автовідповіді клієнтам — бо це та зона, де можна зекономити багато часу, але й легко нашкодити репутації.

**Що добре автоматизується:**
- Відповіді на FAQ (де мій товар? як повернути? яка гарантія?)
- Збір контактних даних і запит на номер замовлення
- Перший контакт у неробочий час — "отримали ваше питання, відповімо вранці"

**Що НЕ варто автоматизувати без нагляду:**
- Скарги та конфліктні ситуації
- Питання про ціни та знижки (бот може пообіцяти те, чого немає)
- Будь-що, де клієнт емоційно заряджений

**Інструменти для чат-ботів з ChatGPT:**
- **Tidio** — є прямий ChatGPT модуль, підключається до сайту за 30 хвилин, є безкоштовний план
- **Crisp** — популярний серед українських магазинів, є AI-модуль
- **Manychat** — якщо основна комунікація через Instagram або Facebook

Для Telegram-магазинів є рішення через BotFather + Make + OpenAI, але це вже рівень 2-3 складності.

## Що перевірити перед запуском

Перш ніж автоматизація пішла "у бій", пройдись по цьому списку:

**Технічна частина:**
- [ ] API ключ OpenAI активований і є баланс на рахунку
- [ ] Встановлені ліміти витрат у налаштуваннях OpenAI (щоб не отримати рахунок на $500)
- [ ] Промпт протестований на 20-30 реальних прикладах перед масовим запуском

**Якість контенту:**
- [ ] Є людина, яка вибірково перевіряє результати (хоча б раз на тиждень)
- [ ] Визначені теми/запитання, де бот НЕ відповідає самостійно
- [ ] Клієнти знають, що спілкуються з ботом (або ні — і ти свідомо обрав цей підхід)

**Юридичне та репутаційне:**
- [ ] Автоматичні відповіді не обіцяють нічого, що ти не можеш виконати
- [ ] Є варіант ескалації до живої людини
- [ ] Опубліковані описи перевірені на відповідність реальним характеристикам товару

## Головне, що варто винести

ChatGPT — це не чарівна кнопка "зробити все само". Це дуже потужний інструмент для рутинних текстових задач, який реально економить час і гроші, якщо правильно налаштований.

Для більшості українських онлайн-продавців найбільша цінність — у двох речах: масова генерація контенту для карток товарів і автовідповіді на типові запитання клієнтів у позаробочий час. Решта — вже залежить від масштабу і готовності інвестувати в налаштування.

Починай з малого: один промпт, одна задача, 10 тестових результатів. Якщо влаштовує — масштабуй.
